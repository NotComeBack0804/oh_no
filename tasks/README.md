# 任务管理

## 目录结构
```
tasks/
├── backlog/           # 待处理任务池
├── in_progress/      # 正在进行
├── testing/           # 测试中
├── completed/        # 已完成
└── README.md
```

## 任务文件命名
`[优先级]_[序号]_[任务简称].md`

例如：`P1_001_用户登录模块.md`
