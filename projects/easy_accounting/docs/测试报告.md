#### 问题 1: 账单列表分类名称显示为空 ✅ 已修复

**修复日期**: 2026-03-23

**修复内容**:
- 在 `Bill.kt` 中新增 `BillWithCategory` 数据类，包含 `categoryName` 字段
- 在 `BillDao.kt` 中新增 3 个 JOIN 查询：
  - `getBillsWithCategoryByDateRange` - 按日期范围查询并 JOIN 分类名
  - `searchBillsWithCategory` - 搜索时 JOIN 分类名
  - `getAllBillsWithCategory` - 全量查询并 JOIN 分类名
- 在 `BillRepository.kt` 中新增对应方法
- 更新 `BillAdapter` 使用 `BillWithCategory` 替代 `Bill`，`bind()` 方法正确显示分类名
- 更新 `MainViewModel.bills` flow 使用 JOIN 查询
- 更新 `MainActivity` 的 `navigateToDetail` 和 `showDeleteDialog` 适配 `BillWithCategory`

#### 问题 2: 数据库加密密钥硬编码 ✅ 已修复

**修复日期**: 2026-03-23

**修复内容**:
- 新增 `SecurityUtils.kt`，使用 Android Keystore 实现密钥安全存储：
  - 通过 `KeyGenerator` 生成 256-bit AES 密钥，存储于 AndroidKeyStore
  - 数据库密码通过 `SecureRandom` 随机生成 32 位字符
  - 密码使用 AES/GCM 加密后存储在 SharedPreferences
  - 首次调用时自动生成并安全存储，后续调用直接读取
- 更新 `AppDatabase.kt`，移除硬编码的 `DATABASE_PASSWORD`，改用 `SecurityUtils.getOrCreateDatabaseKey(context)`

#### 问题 3: 统计图表 Tab 页面初始不可见 ✅ 已修复

**修复日期**: 2026-03-23

**修复内容**:
- 更新 `MainActivity.setupTabs()` 的 `onTabSelected` 回调：
  - Tab 0（支出）: `statisticsContainer.visibility = View.GONE`
  - Tab 1（收入）: `statisticsContainer.visibility = View.GONE`
  - Tab 2（统计）: `statisticsContainer.visibility = View.VISIBLE`
  - 原来缺失的 STATISTICS Tab 的 visibility 设置已补充
