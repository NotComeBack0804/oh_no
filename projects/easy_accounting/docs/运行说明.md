# 简易记账 APP - 运行说明

## 项目概述

简易记账是一款 Android 原生应用，采用 Kotlin 开发，使用 Room 数据库存储数据，支持数据库加密。

## 技术栈

- **语言**: Kotlin
- **最低 SDK**: Android 7.0 (API 24)
- **目标 SDK**: Android 14 (API 34)
- **架构**: MVVM
- **数据库**: Room + SQLCipher 加密
- **UI**: Material Design 3
- **图表**: MPAndroidChart
- **异步**: Kotlin Coroutines + Flow

## 运行要求

1. **JDK 17+**: Android Gradle Plugin 8.x 需要 JDK 17
2. **Android SDK**: 需要 Android SDK 34 (compileSdk)
3. **Gradle**: 8.7+ (通过 gradle wrapper 分发)

## 快速开始

### 1. 安装 JDK 17

如果还没有 JDK 17，请从 [Adoptium](https://adoptium.net/) 下载安装。

### 2. 设置 Android SDK

下载并安装 Android SDK Command Line Tools:
```bash
# Windows
cmdline-tools-win-9477386_latest.zip

# 解压到 D:\Android\SDK\cmdline-tools
# 然后运行以下命令安装必要的组件:
sdkmanager.bat "platforms;android-34" "build-tools;34.0.0" "platform-tools"
```

### 3. 配置 local.properties

在项目根目录创建 `local.properties`:
```properties
sdk.dir=D:\\Android\\SDK
```

### 4. 运行构建

```bash
# Windows
gradlew.bat assembleDebug

# Linux/Mac
./gradlew assembleDebug
```

### 5. 安装 APK

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 数据库

- **数据库名称**: `easy_accounting.db`
- **加密**: 使用 SQLCipher，加密密钥在 `AppDatabase.kt` 中定义
- **版本**: 2
- **默认数据**:
  - 7 个支出分类：餐饮、交通、购物、娱乐、医疗、教育、其他
  - 4 个收入分类：工资、奖金、投资收益、其他
  - 4 个默认账户：支付宝、微信、银行卡、现金

## 阶段二：自动记账功能 (TASK_PM_005 ~ TASK_PM_007)

### TASK_PM_005: Android 无障碍服务
- ✅ `PayAccessibilityService` - 支付监听无障碍服务，监听 `TYPE_WINDOW_STATE_CHANGED` 和 `TYPE_WINDOW_CONTENT_CHANGED` 事件
- ✅ `res/xml/pay_accessibility_service.xml` - 无障碍服务配置，注册支付宝/微信包名
- ✅ `AccessibilitySettingActivity` - 无障碍服务引导设置页面
- ✅ 隐私声明：服务仅读取支付金额，不收集/上传个人信息

### TASK_PM_006: 支付宝/微信自动抓取
- ✅ 正则匹配金额：`¥?(\d+\.?\d{0,2})`
- ✅ 识别页面来源：支付宝 `com.eg.android.AlipayGPhone` / 微信 `com.tencent.mm`
- ✅ `PendingRecord` 表存储待确认账单（id, amount, source, date, created_at, status）
- ✅ 提取成功后自动发送到待确认表

### TASK_PM_007: 待确认账单气泡
- ✅ `FloatBubbleService` - 悬浮窗服务（ForegroundService）
- ✅ 气泡显示：金额、来源（支付宝/微信图标）、时间
- ✅ 分类快捷按钮：餐饮/交通/购物/其他
- ✅ 点击确认 → 保存到 `Bill` 表并更新 `PendingRecord` 状态为 `CONFIRMED`
- ✅ 点击忽略 → 更新 `PendingRecord` 状态为 `IGNORED`
- ✅ 主界面工具栏菜单提供「自动记账设置」入口

### 新增数据表：PendingRecord

| 字段 | 类型 | 描述 |
|------|------|------|
| id | Long (PK, auto) | 主键 |
| amount | Double | 金额 |
| source | PaySource | ALIPAY / WECHAT |
| date | Long | 支付时间戳 |
| createdAt | Long | 创建时间戳 |
| status | PendingStatus | PENDING / CONFIRMED / IGNORED |

### 权限说明

| 权限 | 用途 |
|------|------|
| `SYSTEM_ALERT_WINDOW` | 显示悬浮窗气泡 |
| `FOREGROUND_SERVICE` | 悬浮窗后台服务 |
| `FOREGROUND_SERVICE_SPECIAL_USE` | 特殊用途前台服务 |
| `POST_NOTIFICATIONS` | 发送服务通知 |
| `BIND_ACCESSIBILITY_SERVICE` | 无障碍服务权限 |

### 使用前配置

1. 进入应用 → 点击工具栏「闪电」图标 → 进入「自动记账设置」
2. 点击「开启无障碍服务」→ 在列表中找到「简易记账」并启用
3. 点击「开启悬浮窗权限」→ 允许显示悬浮窗
4. 开启「自动记账」开关

## 构建变体

- **Debug**: `assembleDebug` - 可直接安装到设备
- **Release**: `assembleRelease` - 需要签名配置

## 注意事项

1. 首次运行会自动创建数据库并初始化默认数据
2. 数据库加密密钥硬编码在代码中，生产环境应迁移到安全存储
3. 删除记录需要长按确认
