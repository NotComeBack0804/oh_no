# 简易记账 APP - 运行说明

## 项目概述

简易记账是一款 Android 原生应用，采用 Kotlin 开发，使用 Room 数据库存储数据，支持数据库加密。

## 技术栈

- **语言**: Kotlin
- **最低 SDK**: Android 7.0 (API 24)
- **目标 SDK**: Android 14 (API 34)
- **架构**: MVVM
- **数据库**: Room + SQLCipher 加密
- **UI**: Material Design 3
- **图表**: MPAndroidChart
- **异步**: Kotlin Coroutines + Flow

## 运行要求

1. **JDK 17+**: Android Gradle Plugin 8.x 需要 JDK 17
2. **Android SDK**: 需要 Android SDK 34 (compileSdk)
3. **Gradle**: 8.7+ (通过 gradle wrapper 分发)

## 快速开始

### 1. 安装 JDK 17

如果还没有 JDK 17，请从 [Adoptium](https://adoptium.net/) 下载安装。

### 2. 设置 Android SDK

下载并安装 Android SDK Command Line Tools:
```bash
# Windows
cmdline-tools-win-9477386_latest.zip

# 解压到 D:\Android\SDK\cmdline-tools
# 然后运行以下命令安装必要的组件:
sdkmanager.bat "platforms;android-34" "build-tools;34.0.0" "platform-tools"
```

### 3. 配置 local.properties

在项目根目录创建 `local.properties`:
```properties
sdk.dir=D:\\Android\\SDK
```

### 4. 运行构建

```bash
# Windows
gradlew.bat assembleDebug

# Linux/Mac
./gradlew assembleDebug
```

### 5. 安装 APK

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 数据库

- **数据库名称**: `easy_accounting.db`
- **加密**: 使用 SQLCipher，加密密钥在 `AppDatabase.kt` 中定义
- **版本**: 1
- **默认数据**: 
  - 7 个支出分类：餐饮、交通、购物、娱乐、医疗、教育、其他
  - 4 个收入分类：工资、奖金、投资收益、其他
  - 4 个默认账户：支付宝、微信、银行卡、现金

## 构建变体

- **Debug**: `assembleDebug` - 可直接安装到设备
- **Release**: `assembleRelease` - 需要签名配置

## 注意事项

1. 首次运行会自动创建数据库并初始化默认数据
2. 数据库加密密钥硬编码在代码中，生产环境应迁移到安全存储
3. 删除记录需要长按确认
